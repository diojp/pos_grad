
# pos_grad Bazzard

Projeto Final da Disciplina Programação Web com linguagem de scripts - Unipê


## Instalação

Instale pos_grad com npm

Altere para o seu banco de dados MongoAtlas na variável MONGO_URL no arquivo .env na raiz do projeto.

```bash
  npm i
  npm start
```
